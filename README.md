# MiniTimer

A beautiful, minimal stopwatch for macOS that stays on top and resizes to be tiny.

## Features
- **Always on Top:** Keep your timer visible during meetings or deep work.
- **Resizable UI:** Scale it from standard size down to a tiny, non-distracting square.
- **Modern Design:** Translucent, glass-morphism aesthetic.
- **Auto-Updates:** Get the latest features automatically from GitHub.

## Installation
1. Download `MiniTimer-latest.dmg` from [Releases](https://github.com/sameerbajaj/MiniTimer/releases/latest)
2. Open the DMG and drag **MiniTimer** to Applications
3. **First launch only:** right-click → Open → "Open Anyway"
   (required because the app is not notarized)

## Updates
The app checks for updates automatically on launch and shows a green banner when a new version is available.
